Website Starter
===============

Create directory::

  $ mkdir website-starter

Initialize npm::

  $ npm init
  entry point: (index.js) src/index.js

Install basic dependencies::

  $ npm install webpack --save-dev
  $ npm install babel-loader@5.3.2 --save-dev

  ...

Install dependencies::

  $ npm install bootstrap
